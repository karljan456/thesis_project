# Wine Type & Quality Prediction

A machine learning project that predicts **wine type** (red/white) and **quality score** from physicochemical properties, with an interactive Streamlit web app.

Built as a bachelor's thesis project.

---

## What's Inside

- **EDA** — exploring chemical properties, distributions, and correlations
- **Model Training** — training and comparing multiple classifiers
- **Best Model Selection** — evaluated by cross-validation and key metrics
- **Streamlit App** — interactive UI to predict wine type and quality in real time

## Dataset

UCI Wine Quality Dataset — 11 physicochemical features (acidity, alcohol, sulphates, etc.) for red and white Portuguese *Vinho Verde* wines.

## Stack

`Python` · `Pandas` · `Scikit-learn` · `Matplotlib / Seaborn` · `Streamlit`

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
